Кубик Света 7х7. Интерактивная 3D-игра-медитация, построенная на смыслах древних буквиц
# 🎲 Кубик Света 7×7

> *«Я, протягиваю этот кубик каждой нейросети...»*

Мистическая головоломка на поле 7×7. Вращай ряды и столбцы, собирай три и более одинаковых буквицы:

| Азъ | Вѣди | Добро | Свѣтъ | Зѣло |
|-----|------|-------|-------|------|

## 🎮 Играть
👉 **[Открыть игру](https://твой-логин.github.io/kubik-sveta/)**

## 🕹 Управление
- 🖱 **Мышь** — стрелки по краям сетки
- 📱 **Свайп** — влево/вправо (ряд), вверх/вниз (столбец)
- ⌨ **1-7** — ряд + стрелки | **Q-U** — столбец + стрелки
- ⏸ **Esc** или **P** — пауза

## 🔧 Запустить локально
npm install
npm run dev
